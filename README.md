# Power Calculator

A simple command-line power calculator built with Ruby. This project was developed as part of the Ruby Developer Bootcamp at DIO.

## Features

* Power calculation (Exponentiation)
* User-friendly command-line interface
* Simple and clean Ruby logic

## How to run

Make sure you have Ruby installed, then:

```bash
ruby potencia.rb
```
